游戏定位：塔防类游戏

1.情景引入：
小鲤是一名刚刚高考完的准大学生，他怀揣着激动又忐忑的心情来到了华南理工大学国际校区。在这里，他即将开启四年的闯关打怪之旅。

2.游戏目标：
小鲤将会遇到难度和内容设置不同的四个关卡，对应大学四年。
每一个关卡中，将会有怪兽攻击，小鲤需要在地图上布置防御塔来抵御怪兽攻击。
若最后小鲤的各个指标在阈值以上则存活，成功通关。

3.预开始——测测你的ASTI（Attend School Type Indicator)
用户初次进入游戏后需要填一份问卷，用于测试用户的上学人格。不同的人格将对应不同的存活阈值。
问卷题目：
- 你每周运动的频率是？
	-每天。一天不运动我浑身难受  （身体健康++++）
	-3-4次。有空就动 （身体健康++）
	-1-2次。差不多动一下（身体健康+）
	-无。运动不了一点（身体健康--）

- 如果这一段时间，你有非常多ddl，非常多要做的事情，你的内心os：
	-啊啊啊，那我要赶快开始做，努力做，拼命做 （学业++）
	-压力这么大，那我摆一会好了，手机启动 （心理健康++）

- 上大学后，你想到未来，又想到自己的成绩：
	-我要去更好的学校，拼搏四年，更上一层楼！（学业++）
	-其实到头来都一样，认真学学就好 （心理++）
	-都寒窗苦读12年了，上大学还不玩，更待何时 （心理++，学业--）

- 日常生活中，你：
	-我是友宝女，我需要和很多朋友贴贴 （联结感++++）
	-我需要一定的私人空间，但如果一直没有朋友和我说话，我也会心里空唠唠的 （联结感+）
	-额啊，我不要社交，solo trip才是最好的地球online打开方式 （联结感--）

- 当你受委屈时：
	-唉，是不是我哪里做的不够好，好伤心 （心理健康++）
	-我已经做得很好了，他们指责我是他们的问题 （心理健康--）

- 期末考试前一晚，你更可能：
	-继续冲刺学习 （学业++）
	-保证睡眠，不熬通宵 （身体健康++）
	-和朋友互相打气 （联结感++）
	-看会儿视频放空 （心理健康++）

- 一天安排被打乱时：
	-立刻重新安排计划，不能浪费时间 （学业++）
	-事已至此，去跑步（运动）得了 （身体健康++）
	-找人吐槽，顺便聊聊天 （联结感++）
	-都被打乱了，那先休息吧 （心理健康++）

- 当你连续一周状态都很低迷时：
	-增加学习投入，找到成就感 （学业++）
	-多和朋友见面 （联结感++）
	-规律锻炼和作息 （身体健康++）
	-给自己放假，放空 （心理健康++）

- 当你拿到一份比预期低的作业分数：
	-立刻分析错题，下次拉回来 （学业++）
	-出去跑两圈把情绪甩掉 （身体健康++，心理健康++）
	-找朋友吐槽一下午，再说 （联结感++，心理健康++）
	-告诉自己一次而已，别 PUA 自己 （心理健康++）



哪个阈值最高，就给哪个type，并列的话就并列输出

人物存活指标：
- 身体健康：有运动模式，点击运动模式，植物的攻击力会下降，但是身体健康的指标会上升。如果不点击运动模式，身体健康的指标会匀速下降
- 学业成绩：若有怪物攻击到目标，学业成绩相应下降
- 心理健康 （有社交压力的怪兽，是攻击心理健康这一块）
- 联结感 connection

怪兽：
- 学科：如微积分、线代、电路....  ->学业成绩down
- 科研，大作业 ->心理健康，学业成绩 down
- 社交issue ->联结感down
- 小组作业被压力 ->心理健康down
- 朋友都很忙，没空理我 ->联结感down
击败怪物可以获得货币，用于升级防御塔

防御塔：
- 咖啡
- ai：可升级。豆包-ds-gpt
- 图书馆
- 上课
- b站
| Type | Attack | 备注 |
| :--- | :--- | :--- |
| ai | 360 度扫射，等级越高，杀伤力越强 | 可升级。豆包-ds-gpt |

| 咖啡 | 攻击范围较小，但是伤害值极高 | 咖啡带来短暂的清醒 |


障碍物：
由玩家决定放置在何处，放置障碍物后，怪物需要重新计算路线，不能跨过障碍物。

宝箱：
- 回忆模式：怀念高中，怀念以前的朋友，忧伤 ->所有植物攻击力下降一段时间
- 地狱模式：期中考偷袭，大boss来袭 
- 奖励模式：遇到一个好的机会，被赏识，货币++，每个指标都++（奖励要足够丰富才能吸引大家开宝箱）
- 博弈小游戏：翘课vs点名。成功则获得货币奖励，失败扣除货币
